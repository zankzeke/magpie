Metallic Glass Finder

This web application allows you to search for metallic glass alloys using a machine learning model trained on data from the "Nonequilibrium Phase Diagrams of Ternary Amorphous Alloys" dataset. Provided with a list of elements and the maximum number of components in the alloy, it will compute the probability that it is possible to form a fully-amorphous ribbon with melt spinning for all possible compositions within those bounds (with a 10at% spacing) using the methods described in recent, yet-unpublished work by the Wolverton group. The 10 compositions with the highest probability will be displayed.
